# FutVerse ⚽🏆

**FutVerse** is a premium, world-class football guessing platform built using **React, Vite, TypeScript, and Tailwind CSS v4**. It features unlimited replayability, simulated real-time multiplayer, knockout tournaments, daily challenges, ranked progression, achievements, a complete stats engine, and an administrator federation console.

## 🚀 Key Features

### ⚽ 1. Core Guessing Engine
* **Fuzzy Autocomplete Search:** Typo-tolerant autocomplete begins after typing 2 characters, prioritizing famous recognizable players first.
* **Metric Comparison Matrix:** Displays nationality, current club, league, position, age, shirt number, dominant foot, height, and shared competitions.
* **Comparative Clue Cards:** Features premium color indicators (Emerald for match, Amber for partial, Slate for mismatch) and dynamic directional arrows (younger/older, taller/shorter, shirt number higher/lower).

### 🏆 2. Ranked Multiplayer Arena (Real-Time Simulation)
* **Matchmaking Lobby:** Find opponents in your Elo tier (placements for first 10 games).
* **Best of 3 Rounds:** Both players solve the exact same target player.
* **Live Activity Feed:** Keep an eye on your opponent's real-time guesses as they play!
* **Elo Ratings & Ranks:** Advance from **Newbie** all the way to **GOAT** rank based on matches solved, solve speed, and hint usage.

### 📅 3. Daily Challenges
* **Shared Target:** One hard-selected player globally each day.
* **Streaks & Leaderboards:** Daily streak tracking and specific daily solver rankings.

### 🏛️ 4. Tournament coordinator
* **Knockout Brackets:** Beautiful quarter-final, semi-final, and final brackets.
* **Interactive Matches:** Play your scheduled matches directly or auto-simulate other brackets.

### 🛡️ 5. Federation Console (Admin Panel)
* **Player CRUD:** Add, edit, or remove custom players with complete transfer and competition histories.
* **Tournament scheduling:** Configure custom prizes and auto-advance brackets.
* **Bonus rewards:** Manage Double XP campaigns and featured events.

### 📈 6. Stats & Achievements
* Lifetime tracking of wins, win rates, streaks, average solve speed, and average guesses.
* Unlockable badges (e.g. *Hintless Master*, *UCL Expert*) mapped to gameplay.

## 💻 Setup and Development

1. **Install Dependencies:**
   ```bash
   npm install
   ```
2. **Start Dev Server:**
   ```bash
   npm run dev
   ```
3. **Build Static Site:**
   ```bash
   npm run build
   ```
